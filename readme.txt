Fast and Light Modern look theme.

== Description ==

Fast, light and white looking modern Wordpress theme built from scratch. 
This theme is based on Bootstrap 4.0 framework.
SerioZN is fully responsive and  mobile friendly theme that is suitable for blogs and business sites=

The centered content column and fine-tuned 
typography also makes it perfect for traditional blogs and business sites. 
 You can give your site a personal touch by changing 
the logo, add or remove additional options like search, social media, email and phone icons via Wordpress Customizer.

This theme has built-in support for custom and Google Ads, subtitles and popular pots slider /with free plugins from Wordpress Plugin Repository/.
